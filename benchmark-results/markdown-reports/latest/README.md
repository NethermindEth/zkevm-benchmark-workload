# Latest Profiling Results

This directory contains the most recent profiling results and analysis reports generated by the zkGas profiling framework.

## Current Results

*No results available yet. Run the profiling scripts to generate results.*

## How to Generate Results

1. **Generate Fixtures**:
   ```bash
   ./scripts/generate-gas-categorized-fixtures.sh
   ```

2. **Run Profiling**:
   ```bash
   ./scripts/run-gas-categorized-benchmarks.sh
   ```

3. **Generate Analysis Reports**:
   ```bash
   ./scripts/generate_results.sh --compare --statistics --output benchmark-results/markdown-reports/latest/profiling-results.md
   ```

## File Types

- **`profiling-results.md`**: Comprehensive analysis of all profiling results
- **`execution-analysis.md`**: Focused analysis of execution metrics
- **`proving-analysis.md`**: Focused analysis of proving metrics
- **`statistical-summary.md`**: Statistical analysis and summaries

## Viewing Results

- **In Browser**: Open any `.md` file in a markdown viewer
- **In Terminal**: Use `cat` or `less` to view files
- **In Editor**: Open files in your preferred text editor

## Archiving

When new results are generated, previous results should be archived:

```bash
# Archive current results
mv benchmark-results/markdown-reports/latest benchmark-results/archived/$(date +%Y-%m)

# Create new latest directory
mkdir -p benchmark-results/markdown-reports/latest
```
